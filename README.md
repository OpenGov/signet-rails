# Signet::Rails

A basic Rails wrapper around the "Signet":https://github.com/google/signet gem

Work in progress

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'signet-rails'
```

And then execute:

```bash
$ bundle
```

Or install it yourself as:

```bash
$ gem install signet-rails
```
