h1. Signet::Rails

A basic Rails wrapper around the "Signet":https://github.com/google/signet gem

h2. Installation

Add this line to your application's Gemfile:

<pre><code>
gem 'signet-rails'
</code></pre>

And then execute:

<pre><code>
$ bundle
</code></pre>

Or install it yourself as:

<pre><code>
$ gem install signet-rails
</code></pre>
